Trade-Wind式 オリジナルTCG開発環境「twdev」

Trade-Wind式 オリジナルTCG開発環境「twdev」（以下、本プログラム）は、 オリジナルTCGの開発を支援する目的で作成されました。

インストール

下記の手順（Windows XPの場合）に従います。 Windowsの他バージョン、Linux、Solaris等の場合は適宜読み替えていただければ幸いです。

1. Java 環境のインストールを確認する

スタート　→　プログラム　→　アクセサリ　→　→　コマンドプロンプト　を実行します。
コマンドプロンプト画面が出たら、その場で"java -version"と入力し、エンターキーを押します。

Java環境が正しくインストールされている場合、

java version "1.6.0_01"
Java(TM) SE Runtime Environment (build 1.6.0_01-b06)
Java HotSpot(TM) Client VM (build 1.6.0_01-b06, mixed mode, sharing)

などと表示されます。これが表示されていれば問題ありません。
（表示例はJava JDK 1.6の場合ですが、必要なJREのバージョン・レビジョンについて、 作者の側で確認しておりません。ご了承願います。）

Java環境が正しくインストールされていない場合、

'java'は、内部コマンドまたは外部コマンド、
操作可能なプログラムまたはバッチファイルとして認識されていません。

などと表示されます。この場合、Java Runtime Enviromnent をインストールしていただく必要があります。 Java.comのページからダウンロードしてください。

2. ファイルをコピーする

次に、このディスクの内容をすべてどこかのフォルダにコピーします。

推奨されるフォルダは、ルート直下、マイドキュメント直下など、英語のパスが通る部分です。

3. 環境を整える

以下、推奨する手順ではありますが、必須の手順ではありません。

3-1. CSVファイルを編集する環境

CSVファイルは、メモ帳などでも編集可能ですが、非常に見づらい環境になってしまいます。 そこで、以下に示すような表計算ソフトのいずれかを用いて編集を行うと、編集の効率がぐんとアップします。

  1. Microsoft Office Excel

    マイクロソフト社の誇る、世界最強の表計算ソフト。Windowsが主流になった理由と言っても差し支えないほどの高い機能と操作性を持つ。 ただし、ずば抜けて高い価格がネック。

  2. 三四郎

    ジャストシステム社の表計算ソフト。ATOKの導入を考えているのであれば、幾ばくかのお金を積んで、 Just Suiteに手を出すと付いてくる。 安いものではないが、ATOK + JustSystem PDF Suite のおまけと考えれば悪くない値段。

  3. StarSuite Calc

    Sun Microsystems社の表計算ソフト。日本では、Googleが 「Googleパック」の一部として無料配布をしている他、 ソースネクストが販売している。 機能が豊富で、フリーソフトというのはおいしい。

  4. OpenOffice Calc

    オープンソースで提供されている。フリーソフトとして定評があるが、インストールを行うには技術力が必要。だがそれがいい。

3-2. Java開発環境

本プログラムを利用する際、制限事項や不十分な機能などがある場合、Javaの知識があれば、本プログラムを改造することが可能です。 例えば、本プログラムでは .png ファイルしか出力できませんが、これを .tif ファイルで出力したい、などの場合です。

この場合、Java Development Kit (JDK) を、 Sun Microsystemsのページからダウンロードしてください。

また、JDKを導入する場合、強力なコンソール環境である Cygwin の導入をお勧めいたします。
（参考）インストール方法

制限事項

ここでは、twdev ver.1.0の制限事項を記載します。

  • 出力形式は.pngに限られる。
  • 入力元ファイル、および、出力先フォルダが存在していない場合はエラー出力を表示し、失敗する。
  • CSVデータに、半角「”」、および、半角「，」を含んではならない。
  • CSVデータに含まれる半角スペースは文字列として扱われる。
  • １種類のテキスト文字列について、利用可能な色は一色のみである。
